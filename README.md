# Coffee Shop Website

A clean, modern, and fully responsive single-page coffee shop website built with **pure HTML5 and CSS3** (no JavaScript or frameworks).

### 📄 Pages Included
- **Home** – Hero section, welcome message, featured items
- **About Us** – Story, values, team/mission
- **Menu** – Categorized coffee, drinks, pastries & prices
- **Contact** – Simple contact form + map/location info + social links

### 🎨 Features
- Mobile-first responsive design  
- Clean typography & warm coffee-themed color palette  
- Smooth hover effects and subtle animations (pure CSS)  
- Accessible structure (semantic HTML)  
- Easy to customize colors, menu items, and images

### 🛠️ Technologies
- HTML5  
- CSS3 (Flexbox + Grid + Media Queries)
